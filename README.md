# 📊 Production & Sales Analytics Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Production and Sales Analytics Dashboard** built using Power BI. It provides insights into business performance by analyzing production output, sales trends, and profitability.

---

## 🎯 Objectives

* Track production efficiency and output
* Analyze sales performance across products and regions
* Identify trends and patterns in revenue and profit
* Support data-driven decision-making

---

## 📊 Key Features

* 📦 Production volume analysis
* 💰 Sales and revenue tracking
* 📈 Profit and cost insights
* 🌍 Region-wise and product-wise performance
* 📅 Time-based trend analysis (monthly/yearly)
* 🎛️ Interactive filters and slicers

---

## 🧰 Tools & Technologies

* **Power BI Desktop (.pbix)**
* Data Modeling (Relationships)
* DAX (Data Analysis Expressions)
* Data Visualization

---

## 📂 Project File

* `PRODUCTION_SALES.pbix` → Main dashboard file

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open it using **Power BI Desktop**
3. Use slicers and filters to explore insights


---

## 💡 Key Insights

* Identified top-performing products and regions
* Analyzed gaps between production and sales
* Observed revenue and profit trends over time
